Bootstrap One-Page Website

This project is a simple one-page responsive website created using Bootstrap 5.
It was made as a class practice project to understand Bootstrap layout, components, and responsiveness.

 Features

Responsive Navbar with dropdown and search bar

Hero section using Bootstrap utilities

Card layout with grid system (responsive on all screen sizes)

Bootstrap placeholder loading card

Clean footer section

Fully responsive using Bootstrap classes only

 Technologies Used

HTML5

Bootstrap 5 (CDN)

CSS (Bootstrap utility classes)
